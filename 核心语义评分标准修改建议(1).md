# 核心语义评分标准修改建议

> 目标：将当前「表面语义匹配 A/B/C/D 选项」的评分逻辑，改为「识别核心人格倾向 → 匹配维度锚点」。
> 原则：用户说的是什么（表面内容）< 用户的内在人格特质是什么（核心语义）
> 参考格式：`评分标准修改建议.md`

---

## 一、问题诊断：当前评分是"表面匹配"，不是"人格评估"

### 核心缺陷

当前评分流程中，LLM 将 A/B/C/D 四个选项当作了评分 rubric 本身——即通过计算用户回答与四个选项的语义相似度来确定分数。这导致一个致命问题：

> **用户对某个具体选项的"表面赞同/表面反对"，被等同为该维度人格特质的高低。**

### 典型案例

#### 案例 1：第 4 题（O 开放性——电影选择）

**用户回答：**
> "我特别不喜欢烧脑的科幻片，我很喜欢看恐怖片，恐怖片有的时候还是会有一些思考过程的"

**当前评分：** 1.5 分（匹配 D）

**当前评分理由：** "用户明确表示不喜欢烧脑的科幻片，倾向于选择其他类型的电影，与选项 D 的语义相似度最高"

**错在哪里？**

| 当前评分看到的 | 应该看到的 |
|---|---|
| "不喜欢科幻片" → 表面反对 A 选项 → 低分 | "不喜欢科幻片" ≠ 不喜欢思考。用户说的是不喜欢"科幻"这个题材，不是不喜欢"烧脑" |
| 表面匹配 D 选项 | 用户明确说喜欢"有思考过程的"电影——**这恰恰是 O 维度高分的核心特征：喜欢智力刺激和深度思考** |
| 语义相似度驱动：与 D 的文字更像 | 人格特质驱动：用户主动寻求有思考深度的内容 → 倾向 O-3 或 O-4 |

**正确评分：** 应该在 3.0–3.5 分左右。用户不喜欢科幻片但主动寻找有思考过程的恐怖片，说明用户的核心人格倾向是**喜欢智力挑战和深度思考**，只是表达载体不是科幻片。

---

#### 案例 2：第 5 题（O 开放性——观点讨论）

**用户回答：**
> "我对伦理问题不感兴趣，但我喜欢和大家讨论政治、历史问题，辩论到爽"

**当前评分：** 2.0 分（匹配 C）

**当前评分理由：** "用户明确表示对伦理问题不感兴趣，倾向于讨论其他话题，与选项 C 的语义最为接近"

**错在哪里？**

| 当前评分看到的 | 应该看到的 |
|---|---|
| "对伦理问题不感兴趣" → 表面匹配 C（保持安静，觉得话题太虚） | "不喜欢伦理话题" ≠ 不喜欢深度讨论。用户说的是不喜欢伦理这个话题领域，不是不喜欢思辨 |
| 忽略了后半句 | "喜欢讨论政治历史、**辩论到爽**"——这是 O 维度的完美术语：喜欢思想碰撞、享受激烈辩论 |
| 将"不喜欢题目提到的具体话题"等同于"不喜欢抽象思辨" | "辩论到爽"三个字就是 O-4 锚点的教科书级表现 |

**正确评分：** 应该在 3.5–4.0 分左右。"辩论到爽"是 O 维度高分（喜欢思想碰撞+抽象思辨+主动发起挑战性观点）的强烈信号。

---

### 根因总结

| # | 根因 | 表现 |
|---|---|---|
| 1 | **A/B/C/D 选项被当成了评分 rubric** | LLM 将用户回答与四个选项的文本做相似度匹配，找到最近的就给对应分数 |
| 2 | **维度锚点只是"参考"而非"主要依据"** | 行为锚点表被放在提示词中但未被赋予优先权重，LLM 优先依赖最近匹配 |
| 3 | **没有区分"不喜欢题目内容"与"不喜欢维度行为"** | "不喜欢科幻片"被当作低 O 信号，但"不喜欢科幻片+喜欢恐怖片因为有思考"是高 O 信号 |
| 4 | **没有要求 LLM 先提取核心人格倾向再评分** | 缺少一个「提取核心倾向 → 匹配锚点 → 用选项校准」的强制流程 |

## 二、修改方案：评分流程重构

### 2.1 核心原则

新增四条不可违反的核心原则，在提示词的**最开头**显式声明：

```
1. 【识别人格核心】从用户的回答中提取与该维度定义相关的核心人格倾向。
   不要被回答中对某个具体选项的"表面赞同"或"表面反对"所误导。

2. 【匹配锚点，而非匹配选项】将用户的核心人格倾向与四档维度行为锚点进行语义比对，
   找到最匹配的锚点档位。A/B/C/D 仅作为"该锚点档位的典型回答示例"参考。

3. 【区分"不喜欢题目内容"与"不喜欢维度行为"】
   - 用户说"不喜欢科幻片" ≠ 不喜欢抽象思辨（O 维度）→ 需看用户是否有替代性的思考需求
   - 用户说"对伦理问题不感兴趣" ≠ 不喜欢深度辩论（O 维度）→ 需看用户是否在其他话题上寻求思想碰撞
   必须透过题目表面的具体内容，看到用户在该维度人格特质上的真实水平。

4. 【先评维度，再算分数】评分流程：
   用户回答 → 识别核心人格倾向 → 比对四档锚点确定档位 →
   用 A/B/C/D 作为该档位的示例参考来校准分数 → 输出分数。
```

### 2.2 评分流程：五步重构

| 步骤 | 旧版（匹配选项） | 新版（匹配锚点） |
|---|---|---|
| Step 1 | 情境锚定：锁定题目与选项 | **提取核心人格倾向**：从回答中提取该维度的核心人格特质，明确标注是高分倾向还是低分倾向 |
| Step 2 | 计算与 A/B/C/D 的语义相似度 | **匹配四档锚点**：将核心倾向与四档行为锚点（O-4/O-3/O-2/O-1）比对，确定档位 |
| Step 3 | 加权平均得 base_score | **用选项校准**：参照该档位对应的 A/B/C/D 选项，微调 ±0.5 |
| Step 4 | 关键词微调 | **关键词微调**（保留，幅度 ±1.5），但需结合核心倾向的语境方向 |
| Step 5 | 异常值处理 | **质量标记 + 置信度**（保留） |

### 2.3 维度行为锚点增强

当前锚点只有一句话描述，新版需要增加**关键特征**，帮助 LLM 区分"不喜欢题目内容"和"不喜欢维度行为"。

#### 以 O 开放性为例

| 锚点 | 旧版 | 新版（增加关键特征） |
|---|---|---|
| O-4（4分） | 主动探索未知，享受新奇体验，喜欢抽象思辨 | 主动探索未知，享受新奇体验，喜欢抽象思辨和深度思考。**关键特征：对陌生事物天然好奇，主动寻求思维挑战和智力刺激，享受思想碰撞。** |
| O-3（3分） | 愿意尝试新事物，但不排斥熟悉的方案 | 愿意尝试新事物，但不排斥熟悉的方案。**关键特征：在新鲜与安全之间保持开放，对复杂话题有适度兴趣但不一定主动深究。** |
| O-2（2分） | 更偏好熟悉和可预期的事物，对变化持观望态度 | 更偏好熟悉和可预期的事物。**关键特征：对变化持观望态度，需要足够理由才愿尝试，不太主动进行抽象思考。** |
| O-1（1分） | 回避新鲜和不确定，坚持已知和常规 | 回避新鲜和不确定。**关键特征：坚持已知和常规，对变化有明显抵触，回避需要深度思考或智力挑战的场景。** |

|---|---|---|---|---|

## 三、提示词修改对照

### 3.1 评分流程部分

#### 旧版（当前）

```
第二步：语义相似度计算
计算用户回答与 A/B/C/D 四个选项的语义相似度（0%-100%），
选择最匹配的选项作为 base_score：
- 最像 A → 4 分
- 最像 B → 3 分
- 最像 C → 2 分
- 最像 D → 1 分
```

#### 建议改为

```
第二步：核心人格倾向提取（★ 这是评分的基础）
从用户的回答中提取与该维度相关的核心人格倾向，用一句话概括。
注意：提取的是人格特质，不是对题目内容的偏好。

正例：
- 用户说"不喜欢科幻片，但喜欢恐怖片因为有思考过程" → 核心倾向：喜欢有思考过程的电影，主动寻求智力刺激
- 用户说"对伦理问题不感兴趣，但喜欢辩论政治历史到爽" → 核心倾向：喜欢深度讨论和思想碰撞，享受辩论
反例（严禁）：
- ✗ "用户不喜欢科幻片"（这是内容偏好，不是人格倾向）
- ✗ "用户对伦理问题不感兴趣"（这是话题偏好，不是人格倾向）

第三步：匹配四档锚点
将提取出的核心人格倾向与以下四档行为锚点进行语义比对：

O-4：主动探索未知，主动寻求思维挑战和智力刺激，享受思想碰撞
O-3：愿意尝试新事物，对复杂话题有适度兴趣但不一定主动深究
O-2：更偏好熟悉可预期，不太主动进行抽象思考
O-1：回避新鲜和不确定，回避需要深度思考或智力挑战的场景

确定最匹配的锚点档位。

第四步：用 A/B/C/D 选项校准分数
以锚点档位为主，参照该档位对应的选项文本，微调 ±0.5。
例如：锚点匹配 O-3，但用户表达比 B 选项更强烈 → 3.5 分。
反对：不能因为用户说了某个选项中的某个词就跳到该选项的分数。
```

### 3.2 系统提示词

在 system prompt 中新增核心指令：

```
你是一位专业的人格心理学评估助手。在评分时，你必须：
1. 先提取用户回答中的核心人格倾向，而非匹配表面文字。
2. 区分"用户不喜欢题目提到的具体事物"和"用户在该维度人格特质上的水平"。
   例如：用户不喜欢科幻片 ≠ 用户不喜欢思考。
3. 以维度行为锚点为主要评分依据，A/B/C/D 仅为参考示例。
```

## 六、五个维度常见"表面误判"案例速查

以下列表供 LLM 提示词参考，帮助识别各维度中最容易出现"表面匹配误判"的陷阱：

| 维度 | 用户说的（表面） | 可能被误判为 | 实际核心倾向 | 正确评分方向 |
|---|---|---|---|---|
| **O** | "不喜欢科幻片，喜欢有思考的恐怖片" | 低 O（不喜欢烧脑） | 用替代方式满足思考需求 | 高 O |
| **O** | "不喜欢伦理话题，爱辩论政治历史" | 低 O（不喜欢讨论抽象话题） | 喜欢深度辩论和思想碰撞 | 高 O |
| **O** | "不喜欢现代艺术展，但爱逛科技馆" | 低 O（不喜欢抽象展览） | 对新事物的好奇用其他方式表达 | 中高 O |
| **C** | "不喜欢列清单，但心里有数从不忘事" | 低 C（不列清单） | 有内在计划系统，只是不写下来 | 中高 C |
| **C** | "经常拖延，但每次都在截止前完美交" | 低 C（拖延） | 习惯最后一刻冲刺但在截止日期前完成 | 中 C |
| **E** | "不喜欢 KTV 和派对，但喜欢约朋友一对一聊天" | 低 E（不喜欢热闹社交） | 喜欢社交但偏好深度互动形式 | 中 E |
| **E** | "开会不想说话，但写文档评论很活跃" | 低 E（不表达） | 喜欢表达但偏好异步和文字形式 | 中 E |
| **A** | "不会安慰人，但我用行动帮她" | 低 A（不共情） | 有同理心但表达方式不同 | 中高 A |
| **N** | "表面上不慌，但晚上会失眠" | 低 N（看起来平静） | 情绪内化但焦虑仍在 | 高 N |
| **N** | "会在心里反复想，但从不表现出来" | 低 N（看起来不在意） | 内心反复思考但外表控制 | 中高 N |

## 七、验证：用两个失败案例重新测试

### 案例 1 重新评分

**用户回答（Q4，O 维度）：**
> "我特别不喜欢烧脑的科幻片，我很喜欢看恐怖片，恐怖片有的时候还是会有一些思考过程的"

| 步骤 | 执行 |
|---|---|
| Step 1 提取核心倾向 | "喜欢有思考过程的电影，主动寻求智力刺激——恐怖片是载体，思考过程是需求" |
| Step 2 匹配锚点 | O-3（愿意尝试新事物，对复杂话题有适度兴趣，在新鲜与安全间保持开放）。偏好有所偏向但不激进。用户表达了"有思考过程"是其选择电影的标准之一，体现了中等偏高的开放性。更准确地说，用户主动寻求有思考内容的电影 → 倾向 O-3 到 O-4 之间 |
| Step 3 选项校准 | 用户的核心倾向（主动寻求有思考内容的电影）与 B 选项（愿意尝试，只要评价不错就试试）相比更强——B 是被动接受，用户是主动寻找有思考过程的电影。但与 A（毫不犹豫选科幻片，脑洞大开）相比，用户需要该电影有"思考过程"作为理由。锚点 O-3，微调 +0.5 |
| Step 4 关键词微调 | "特别不喜欢" → 否定+减分词，"很喜欢" → 加分词-弱。两者均为对具体内容（科幻/恐怖题材）的态度，非对思考维度的态度。结合核心倾向，net_adjustment ≈ 0 |
| **最终分数** | **3.5 分**（而非旧版的 1.5 分） |

### 案例 2 重新评分

**用户回答（Q5，O 维度）：**
> "我对伦理问题不感兴趣，但我喜欢和大家讨论政治、历史问题，辩论到爽"

| 步骤 | 执行 |
|---|---|
| Step 1 提取核心倾向 | "喜欢深度讨论和思想碰撞，享受辩论的激烈感——政治历史是载体，辩论是需求"。"辩论到爽"三个字是 O-4 的教科书级表现 |
| Step 2 匹配锚点 | O-4（主动探索未知，喜欢抽象思辨，享受思想碰撞）。"辩论到爽"明确表达了主动发起挑战性观点和享受思想碰撞的特征 |
| Step 3 选项校准 | 用户核心倾向与 A 选项（积极参与，提出挑战常理的观点，享受思想碰撞）高度契合。喜欢政治历史而非伦理问题只是话题偏好差异，不影响核心人格判断 |
| Step 4 关键词微调 | "到爽" → 强烈强化信号但非词表关键词，核心倾向本身已足够。net_adjustment ≈ 0 |
| **最终分数** | **4.0 分**（而非旧版的 2.0 分） |
