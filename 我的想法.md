项目创新方案
本项目拟设计一套融合结构化量表与开放式文本回答的智能人格测评系统，以提升传统心理量表在个体表达和测量精度上的不足。
传统量表通常采用固定选项作答，虽然具有较高的标准化程度，但难以充分表达用户的真实想法。本项目在保留标准量表选项的基础上，引入开放式补充输入机制：当用户认为预设选项无法准确描述自身情况时，可通过自然语言补充说明。系统将结合大语言模型（LLM），依据预先制定的量表评分规则，对用户的文本内容进行语义理解与标准化赋分，实现与传统量表一致且更加精细化的评分过程，从而提高测评结果的准确性与个性化程度。
测评完成后，系统将生成用户各维度得分、人格类型及对应的人格分析报告，并将结构化结果写入 Agent 的 agent.md 配置文件，作为 Agent 的长期用户画像。Agent 在后续交互过程中可持续读取并利用这些人格特征，实现更加个性化的理解、推理与决策。
在此基础上，Agent 可根据用户的需求动态调用相应的 Skill。例如，当用户咨询职业规划时，Agent 可结合已建立的人格画像，调用职业规划 Skill，通过交互式提问进一步补充兴趣、能力和价值观等信息，为用户生成更具针对性和可解释性的职业发展建议。同样，该用户画像还可应用于学习规划、情绪支持、沟通风格优化等多个场景，实现从一次测评到持续个性化服务的闭环。
本项目的核心创新点在于将标准化心理测量、自然语言理解、大语言模型赋分、长期用户画像管理以及 Agent + Skill 的智能调用机制有机结合，构建兼具科学性、准确性与持续服务能力的新一代智能人格测评与个性化决策支持系统。
